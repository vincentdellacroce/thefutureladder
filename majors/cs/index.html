💻 Computer Science

"The discipline that runs the modern world."
What is CS?
Build the World's Tools.

Computer Science is the study of computation, algorithms, data structures, and systems. CS graduates build software, design AI systems, secure networks, and create the products billions of people use every day. It is the highest-paying undergraduate degree in America — and has been for a decade.

Unlike engineering, CS is highly portable: you can work in any industry (finance, healthcare, media, government) in any city. You can also work remotely, start your own company, or pivot into almost any adjacent field. The combination of high pay, flexibility, and demand makes CS arguably the best major in America right now.

The reality: CS is competitive — admission to top CS programs (MIT, CMU, Stanford, UIUC, Berkeley) is extremely selective. But mid-tier schools like Purdue, UMass, or UMD produce excellent engineers who get hired at top companies. Your GitHub portfolio often matters more than your school name.

High School Roadmap

  • 9th: Start learning Python. Codecademy, freeCodeCamp. Build your first small project.
  • 10th: AP CS Principles. Start competitive programming (USACO Bronze). Build a real web app or tool.
  • 11th: AP CS A (Java). USACO Silver push. Contribute to open source or start a real project. Intern if possible.
  • 12th: AP Calc BC. Build a portfolio of 3–5 substantial projects. USACO Gold if possible. Apply.

What CS Students Actually Do in College

  • Algorithms and data structures (the core)
  • Systems programming in C/C++
  • Computer architecture and OS
  • Theory of computation and discrete math
  • Electives: AI/ML, databases, networking, security
  • Large project-based courses and team software builds
AP Courses
Build the Technical Foundation

🔴 Essential

AP CS A (Java) AP Calc BC AP Statistics
  • AP CS A — Java is the language of this course. Teaches OOP, recursion, arrays, ArrayList. Essential baseline. Aim for a 5.
  • AP Calc BC — Discrete math and algorithm analysis require calculus intuition. Required for most CS programs.
  • AP Statistics — Machine learning and data science are deeply statistical. Build this foundation early.

🟡 Highly Recommended

AP CS Principles AP Physics C AP Micro/Macro
  • AP CS Principles — Good intro before AP CS A. Lighter but useful for conceptual grounding.
  • AP Physics C — Shows quantitative strength. Important for hardware/systems engineers.
  • AP Economics — Many CS grads become entrepreneurs or product managers. Business literacy matters.
Extracurriculars
Build Things. Compete. Publish.
The most important thing: Build projects and put them on GitHub. A high school student with 5 real projects on GitHub signals more to a CS admissions committee than any club membership.

🏆 Tier 1

  • USACO — USA Computing Olympiad. Gold/Platinum level is extremely impressive. Bronze/Silver is still great. Start early.
  • Google Code-in / GSOC — Google Summer of Code for open source. Massive credential.
  • Software internship — Intern at any tech company or startup. Real experience beats school projects.
  • Published app with real users — Ship something to the App Store or web. 1,000 downloads means more than most clubs.
  • Research publication — Work with a professor on ML or systems research. Rare and extremely impressive.

🥈 Tier 2

  • Hackathons — MLH hackathons, local/national events. Winning is great; participating shows hustle.
  • Coding Club / President — Run workshops, teach others, build club projects
  • Science Olympiad — Programming events signal competitive algorithmic thinking
  • Personal GitHub portfolio — 5+ projects with clean documentation. This is your technical resume.
  • Competitive math (AIME) — Discrete math and algorithm reasoning overlap significantly

🛠 Projects to Build (Tier by Difficulty)

  • Simple: calculator, to-do app, personal website
  • Medium: weather app (API), Discord bot, game clone
  • Hard: ML model, full-stack web app, compiler or interpreter
  • Elite: Something novel with real users. Anything that solves an actual problem.

Specializations
Which Track in CS?
🧠 Artificial Intelligence / ML
$130K → $350K+ (senior)
The hottest field in tech. Machine learning, deep learning, NLP, computer vision. Requires strong math (linear algebra, probability). Employers: Google DeepMind, OpenAI, Meta AI, Anthropic.
🌐 Software Engineering (SWE)
$95K → $280K+ (senior)
The most common CS career. Build products, apps, and systems at scale. Frontend, backend, full-stack. Every industry needs SWEs. FAANG pays the highest.
🔐 Cybersecurity
$85K → $185K+ (senior)
Penetration testing, threat intelligence, security engineering, cryptography. Massive government and private sector demand. Growing fast. DEF CON and CTF competitions are your ECs.
📊 Data Science / Analytics
$90K → $200K+ (senior)
Extract insights from data. Python, SQL, statistics are your tools. Works in every industry. Distinct from ML but overlapping — data engineers are in extreme demand.
⚙️ Systems / Infrastructure
$100K → $280K+ (senior)
Operating systems, compilers, distributed systems, databases. Lower profile but extremely high-paying. Google SRE, systems engineers at Jane Street, and kernel engineers at Microsoft/Apple are elite.
💹 Quantitative Finance / HFT
$150K → $500K+ (senior)
Apply CS to trading. High-frequency trading firms (Jane Street, Citadel, Two Sigma, DE Shaw) are the highest-paying employers of CS/math graduates. Requires exceptional algorithmic thinking.
🎮 Game Development
$70K → $150K (senior)
Game engines, graphics, physics simulation. Lower pay than other CS tracks but extremely creative. Unity, Unreal, Roblox are common tools. Hard to break into at AAA level.
🏥 Health Informatics / Bioinformatics
$80K → $165K (senior)
Apply CS to medicine and biology. Genomics, EHR systems, medical imaging AI. Growing rapidly with funding from NIH and pharma. Great dual-path for CS + pre-med interests.
Resources
What to Use, Read, and Watch
Competition
USACO
USA Computing Olympiad. Start at Bronze. The #1 CS competition for high schoolers. Train here for interviews too.
Free Course
Harvard CS50
The most popular intro CS course in the world. Free online. Covers C, Python, web, SQL. Take this before AP CS A.
Practice
LeetCode
Algorithm practice. Big tech interviews are all about this. Start with Easy problems, then Medium. Practice daily.
Platform
GitHub
Host your code here. Build a portfolio. A clean GitHub profile is your CS resume. Start now, push consistently.
Free Learning
freeCodeCamp
Free full curriculum: HTML/CSS, JavaScript, Python, data science. Earn certificates. Excellent self-study path.
Competition
Codeforces
The premier competitive programming platform. Train for USACO and interviews simultaneously. Contest ratings are respected.
Free Course
MIT: The Missing Semester
What CS programs don't teach: shell, git, editors, debugging. Essential practical skills. Free from MIT CSAIL.
Interview Prep
NeetCode
Structured LeetCode roadmap with video explanations. The best resource for technical interview prep. Start in 11th grade.
📚 Must-Read Books
Book
The Pragmatic Programmer
Career advice and mindset for software engineers. Read this before you start college. Shifts how you think about code.
Book
Cracking the Coding Interview — McDowell
The bible of technical interviews. 189 problems with solutions. Start practicing this in 11th grade.
Book
Structure and Interpretation of Computer Programs (SICP)
MIT's legendary textbook. Dense but builds deep understanding of computation. Free online at mitpress.mit.edu.
